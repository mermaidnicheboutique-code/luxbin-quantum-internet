## Step 2: After Form Submission

### Create CMC Account

1. **Sign Up:** https://accounts.coinmarketcap.com/signup
2. **Use same email** as project contact
3. **Verify email address**
4. **Complete profile**

### Claim Your Listing

Once CMC reviews and lists your token:

1. **Log in to CMC account**
2. **Go to your token page**
3. **Click "Claim this project"**
4. **Complete verification process:**
   - Prove ownership via social media
   - Or sign message from contract deployer wallet
5. **Get verified badge ‚úì**

- [ ] Circulating supply (monthly as burns occur)

## Common Issues & Solutions

### Issue: Application Rejected - Insufficient Data
**Solution:**
- Add more trading volume naturally
- Increase liquidity
- Build larger community
- Wait longer before reapplying (30 days)

### Issue: Circulating Supply Unclear
**Solution:**
- Provide clear calculation method
- Create API endpoint for real-time supply
- Document all locked/burned tokens
- Show proof of locks

### Issue: Suspicious Activity
**Solution:**
- Remove wash trading
- Build organic community
- Remove bot followers
- Be transparent about all aspects

### Issue: Missing Information
**Solution:**
- Double-check all form fields
- Ensure all links work
- Add comprehensive documentation
- Provide contact information
