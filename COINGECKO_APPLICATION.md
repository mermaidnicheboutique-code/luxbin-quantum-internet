## Pre-Submission Checklist

Before submitting to CoinGecko, ensure:

### Smart Contract Requirements
- [ ] Contract verified on BaseScan (https://basescan.org)
- [ ] Contract is not a proxy or upgradeable (or clearly documented)
- [ ] Source code is publicly viewable
- [ ] No major security vulnerabilities

### Trading Requirements
- [ ] At least one DEX listing (Uniswap on Base)
- [ ] Minimum $1,000 liquidity in pool
- [ ] Trading volume for at least 24 hours
- [ ] Price discovery has occurred (not just launch price)

### Information Requirements
- [ ] Working website or comprehensive GitHub README
- [ ] Active social media (Twitter with 100+ followers minimum)
- [ ] Community channel (Telegram or Discord with 50+ members)
- [ ] Clear project description and documentation
- [ ] Whitepaper or detailed tokenomics doc

### Marketing Requirements
- [ ] Logo file ready (200x200 PNG, transparent background)
- [ ] Banner image (1200x600 recommended)
- [ ] Project description written (short and long versions)
- [ ] All social links active and updated

## After Submission

**1. Tracking:**
- You'll receive a confirmation email
- Check spam folder if not received
- CoinGecko review typically takes 1-2 weeks

**2. Follow Up:**
- If no response after 2 weeks, email: submissions@coingecko.com
- Reference your submission details
- Be polite and patient

**3. Once Listed:**
- [ ] Verify all information is correct
- [ ] Update your website with "Listed on CoinGecko" badge
- [ ] Announce listing on social media
- [ ] Request community to add to watchlists
- [ ] Encourage holders to vote on CoinGecko

**4. Ongoing Maintenance:**
- [ ] Update circulating supply monthly (as burns occur)
- [ ] Update social links if changed
- [ ] Respond to community questions on CoinGecko page
- [ ] Submit updates for major milestones

## Common Rejection Reasons & Solutions

**Rejection: Insufficient trading activity**
- Solution: Wait for more organic volume, add more liquidity

**Rejection: Incomplete information**
- Solution: Ensure all fields filled, all links working

**Rejection: Suspicious project**
- Solution: Add team information, lock liquidity, verify contract

**Rejection: Duplicate submission**
- Solution: Wait for original submission review, don't reapply

**Rejection: Not meeting criteria**
- Solution: Review CoinGecko methodology, improve metrics
